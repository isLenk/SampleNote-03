Launcher Form

The first window that will appear upon startup is the launcher form.
Unlike the SampleNote v2. Launcher, everything required to load up compatible spreadsheet files.

Controls

  1. Recent Files - Displays the latest available files that were used
  2. New File - Generates a new formatted spreadsheet document
  3. Load File - Attempt to load spreadsheet documents
  4. Selected API - Change custom API for spreadsheet program.
    (only custom written '.dll' based)
  5. Exit Form
